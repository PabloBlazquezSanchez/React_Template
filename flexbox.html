Diseño de Caja Flexible en CSS

El Módulo de Caja Flexible fue diseñado como un modelo unidimensional de layout,
y como un método que pueda ayudar a distribuir el espcio entre los objetos de una interfaz
y mejorar las capacidades de alineación.

Antes del módulo de diseño Flexbox, había 4 modos de diseño:

  • En bloque, para secciones de una página web
  • En línea, para texto
  • Tabla, para datos de tablas bidimensionales
  • Posicionado, para la posición explícita de un elemento
Este módulo de diseño facilita el diseño de una estructura de diseño flexible y
receptiva sin usar flotación o posicionamiento.

Elemento principal (Contenedor)

Para comenzar a usar el modelo flexbox, en primer lugar debemos definir un contenedor flexible.
Las propiedades del contenedo flexible son:

  • Dirección flexible
  • Envoltura flexible
  • Flujo flexible
  • Justificar el contenido
  • Alinear elementos
  • Alinear contenidos
Para que el contenedor se vuelva flexible le tenemos que dar el estilo display:flex;.

Los elementos secundarios directos de un contenedor flexible se convierten en elementos flexibles.
Las propiedades del elemento flexible son:

  • Orden
  • Crecimiento flexible
  • Flexión-encogimiento
  • Base flexible
  • Flexionar
  • Alinearse
